# Day 2: ES6 Features
In this project, we will explore some of the key features introduced in ECMAScript 6 (ES6), which is a major update to the JavaScript language. ES6 includes a variety of new syntax and functionalities that make it easier to write cleaner and more efficient code.
## Features Covered
1. **Arrow Functions**: A shorter syntax for writing functions.
2. **Let and Const**: New ways to declare variables with block scope.
3. **Template Literals**: A new way to create strings that allows for embedded expressions.
4. **Destructuring Assignment**: A convenient way to extract values from arrays or objects.
5. **Default Parameters**: A way to set default values for function parameters.
6. **Spread Operator**: A syntax for expanding arrays or objects.
7. **Classes**: A new syntax for creating objects and handling inheritance.
8. **Promises**: A way to handle asynchronous operations.
